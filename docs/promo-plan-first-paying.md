## 1. Математика цели
```
10 платящих
  ← ~130–160 стартов триала        (конверсия триал→оплата ~6–8%)
  ← ~1000–1500 целевых визитов      (CR лендинга ~10–15%)
  за 8–10 недель → ~120–180 визитов/нед
```
В действиях: **2 контент-публикации/нед + присутствие в 2–3 сообществах + ASO**.

## 6. Конверсия триала (email + push)
Цепочка по дню триала: день 0 «создай участок» → 1 «что подсказывает приложение» → 5 «твой сезон в
цифрах» → 6 «триал кончается + оффер» → 8 win-back с промокодом. Связать с paywall день 5–6.
⚠️ **Блокер: доставляемость.** Письмо-чек улетело в спам (19.06). Без **SPF/DKIM/DMARC** цепочка
пойдёт в спам и конверсия умрёт. Технический приоритет №1 перед нагоном трафика.

## 7. Реферальная петля
Каждому платящему — промокод «подари другу 2 недели Про» (`promo_codes` есть, погашения трекаются).
Цель — >0.3 погашения на платящего.

## 8. Метрики (еженедельно, авто)
Визиты лендинга → старты триала → оплаты → погашенные промокоды (`/statistic_funnel` на авто-отчёт).
**Пороги:** CR триал→оплата <5% → чинить онбординг/активацию; визитов <100/нед → добавить тему/канал.

## 10. Риски
- **Письма в спам** → DKIM/SPF/DMARC (приоритет №1).
- **Сезон уходит** (пик весна-июнь) → сейчас «уход/полив/урожай/заготовки», в августе-сентябре
  «подзимние посевы» и «планирование сезона-2027», задел контента к весеннему пику.
- **Нет авторитета** → только value-first, без спама; быстрее собрать первые отзывы.
- **Внешние ссылки режут охват ВК/Дзен** → ссылка в кнопку сообщества / первый комментарий / конец поста.
- **Мало времени** → один мотор (Дзен) ведём хорошо, остальное по остаточному; авто-постер и
  авто-отчёты экономят часы.

## Приложение. Зеркало в ВК
ВК режет охват постам с внешними ссылками в теле → ссылку класть в кнопку сообщества / первый
комментарий / конец поста.
- **Настройки:** стена «Ограниченная», комментарии вкл; обложка 1590×400; аватар = иконка; кнопка
  действия «Перейти на сайт» → `https://dacha.studio1008.com`; закреп = приветственный пост.
- **Адаптация поста:** крючок в первых 2–3 строках; картинка нативной загрузкой (не превью ссылки);
  3–5 хэштегов (`#дача #огород #сад #рассада #урожай`); ссылка — в первый комментарий.
- **Кросспост:** на старте руками (2 поста/нед — дешевле, чем интеграция). Авто через VK API
  (`wall.post` с токеном сообщества) на Node-бэкенде — когда выйдем на 4+ постов/нед.
