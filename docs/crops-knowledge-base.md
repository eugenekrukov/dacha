## Структура данных в БД

### Схема расширения (миграция 005)
```sql
crops:
  climate_zones       JSONB  -- Зоны 3-6 (USDA), sow_start/end, transplant_start/end (день года)
  watering_details    JSONB  -- По стадиям: seedling/sprouted/growing/flowering/fruiting
  fertilizing_schedule JSONB  -- [{stage, timing, fertilizer_type, product_example, dose, method, notes}]
  diseases            JSONB  -- [{name, symptoms, conditions, treatment, prevention}]
  pests               JSONB  -- [{name, signs, treatment, prevention}]
  good_neighbors      TEXT[] -- Совместимые культуры (по имени)
  bad_neighbors       TEXT[] -- Несовместимые культуры
  good_predecessors   TEXT[] -- Хорошие предшественники
```

### Климатические зоны (ключи)
| Зона | Регионы | Последний заморозок |
|------|---------|---------------------|
| `"3"` | Сибирь, Дальний Восток | ~5-10 июня |
| `"4"` | Урал, Южная Сибирь | ~25 мая - 5 июня |
| `"5"` | Средняя полоса, Северо-Запад | ~10-25 мая |
| `"6"` | Юг РФ, Черноземье, Поволжье | ~10-30 апреля |

### Стадии роста (plantings.stage)
`sowing` → `sprouted` → `growing` → `flowering` → `fruiting` → `harvesting` → `done`

## Схемы подкормок по типам культур

### Плодовые овощи (томат, перец, баклажан, огурец)
```
Рассада (нед. 2)  → N  (нитрофоска 10г/10л)
Через 2 нед. после высадки → NPK (Кемира 20г/10л)
Цветение          → PK  (монофосфат калия 15г/10л)
Плодоношение      → K   (сульфат калия 20г/10л, каждые 10-14 дней)
```

### Корнеплоды (морковь, свёкла, редька)
```
Фаза 3-4 листьев  → N   (аммиачная селитра 10г/10л)
Июль              → NPK (нитрофоска 20г/10л)
Август            → K   (сульфат калия / зола)
```

### Капустные
```
Рассада фаза 3-4 л → N   (аммиачная селитра)
Через 2 нед. после высадки → N
Завязывание кочана → PK  (снизить N)
Обязательно добавлять Бор 2г/10л для цветной и брокколи
```

### Ягодные культуры (многолетники)
```
Ранняя весна      → N   (аммиачная селитра 15г/м²)
После плодоношения → PK  (суперфосфат + сульфат калия по 20г/м²)
```

## Ключевые болезни и вредители по культурам

### Фитофтороз (томат, картофель, перец)
- **Условия**: влажность >75%, перепад температур
- **Препараты**: Ридомил Голд, Превикур Энерджи, бордоская смесь 1%
- **Профилактика**: обработка с начала июля каждые 10-14 дней

### Мучнистая роса (огурец, кабачок, крыжовник, горох)
- **Условия**: жара + влажность, или сухость после дождя
- **Препараты**: Топаз, Квадрис, коллоидная сера, сода+мыло
- **Профилактика**: нормировка, проветривание

### Серая гниль (клубника, томат, огурец)
- **Условия**: влажность >80%, загущённость
- **Препараты**: Свитч, Фундазол, Трихоцин
- **Профилактика**: мульчирование, нижний полив

### Кила (капустные)
- **Условия**: кислая почва pH < 6,5
- **Лечение**: нет
- **Профилактика**: известкование до pH 7, севооборот 5-7 лет

### Колорадский жук (картофель, баклажан)
- **Препараты**: Командор, Актара, Конфидор; биометод — Бовергрин
- **Профилактика**: мульча соломой, посадка хрена по периметру

### Паутинный клещ (огурец, перец, малина)
- **Условия**: сухость, высокая температура
- **Препараты**: Фитоверм, Акарин, Клещевит
- **Профилактика**: влажность воздуха >70%, опрыскивание листьев снизу
