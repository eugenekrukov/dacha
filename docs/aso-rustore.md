```
Установите «Календарь дачника» — и ваш сад и огород всегда под контролем!

```

## ASO-логика

- **Главный ключ в первом предложении** полного описания — RuStore придаёт вес началу текста.
- **Высокочастотные запросы** распределены естественно: «сад», «огород», «дача», «рассада»,
  «лунный календарь», «когда сажать», «теплица», «полив», «подкормка», «урожай».
- **Сканируемость**: эмодзи-маркеры и буллиты → выше конверсия установки (поведенческие
  метрики тоже влияют на ранжирование).
- **Блок «Кому подойдёт»** ловит запросы «для дачников / для садоводов / для начинающих».
- **Строка «Темы»** в конце — крючок для индексации длинного хвоста, не мешает чтению.

## Чувствительные разрешения (для модерации)

В декларации указаны и обоснованы:

```
ACCESS_COARSE_LOCATION — определение населённого пункта участка для прогноза погоды и агро-рекомендаций.
ACCESS_FINE_LOCATION — точная привязка погоды и расчёта работ к координатам участка пользователя.
POST_NOTIFICATIONS — напоминания о дачных работах (полив, подкормка) и оповещения о заморозках и жаре.
```

**Типы данных** (RuStore «Данные приложения»): местоположение (приблизит./точное),
адрес эл. почты, ID пользователей, ID устройства (push-токен), другой пользовательский контент,
история покупок (статус подписки). «Имя» больше не собирается.
