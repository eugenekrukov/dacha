# Тема: Лента «Мой участок»

> обновлено: 2026-06-28, commit `1dc79dd`

Сквозная фича через все три платформы — реализована и задеплоена.

## Суть
Вкладка «Инфо» заменена на «Профиль / Мой участок» (табы Лента / Статистика / Справочник). Сводная приватная лента всех фото + вехи сезона (посев/высадка/первый урожай/завершение), группировка по месяцам. Запись-центричная модель (`action`/`photo`/`milestone`), единый блок «действие+заметка+фото».

## Backend
`GET /feed` (UNION, пагинация, без миграции). См. [[02 Backend]].

## Web
Паритет с Android по разделу «Профиль» — задеплоено 2026-06-23 (`e5913fe`). См. [[03 Web]].

## Android
Реализована, влита в `main` (`baa79e9`, 2026-06-22), ⚠️ не запушена/не задеплоена на момент записи.
