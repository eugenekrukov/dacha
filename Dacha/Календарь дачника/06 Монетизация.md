# Монетизация (финал 2026-06-16)

Web + Google Play + RuStore — **все платная подписка «Дачник Про», 7 дней триал, без рекламы.** Samsung снят с публикации.

## Оплата
ЮKassa напрямую (Shop ID 1376599), **разовые платежи** (рекуррент запрещён самозанятому → продление вручную). Доступ продлевает вебхук `/billing/webhook`.

## Чек НПД (422-ФЗ)
Авторегистрация в «Мой налог» через `lknpd.nalog.ru` (сервис ЮKassa «Чеки для самозанятых» прекращён 29.12.2025).

Поток: `webhook → payments.npd_status → cron nalogJob → addIncome через RU-релей (nalog-relay.php на reg.ru) → письмо с чеком`.

- `YOOKASSA_RECEIPT_MODE=off`
- ✅ E2E подтверждён реальным платежом 19.06.2026
- Релей терял токен ФНС (Apache режет `Authorization` → токен теперь в `X-Relay-Auth`)
- Секрет релея — в файле `.relay-secret` на сервере, НЕ в гите; `NALOG_RELAY_SECRET` ротирован
- reg.ru shared хостинг блокирует SSH-форвардинг — релей развёрнут как обычный PHP-файл

Детали: `docs/DEPLOY.md` (раздел «Мой налог»), `session-note.md` (2026-06-19).

## Доступ
