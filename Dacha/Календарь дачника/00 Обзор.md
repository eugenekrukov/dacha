# Календарь дачника — обзор проекта

> ⚠️ Источник истины — репозиторий `C:\Projects\Dacha\Календарь дачника`. Эта база — навигация + факты, которых нет в репо. При конфликте верь репо.

## Что это
Мобильное + веб-приложение для дачников: календарь работ, журнал посадок/действий, погода, рекомендации, справочник проблем растений. MVP завершён, в проде, идёт пост-MVP-разработка.

## Стек
- **Backend**: Node.js 20 + Fastify 4 + PostgreSQL
- **Web**: React + Vite + TypeScript + Tailwind
- **Android**: Kotlin + Jetpack Compose + Hilt, флейворы `rustore`/`gplay` (samsung удалён 2026-06-30)

## Прод
- Backend: https://dacha.studio1008.com/ (pm2 `dacha-api`)
- Web: https://dacha.studio1008.com/app/
- Android package: `ru.dachakalend.app`
