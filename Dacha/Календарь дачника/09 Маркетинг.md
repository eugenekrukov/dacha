# Маркетинг

Цель: запуск РФ, аудитория 35+, 0 бюджета, ≥10 платящих за 2–3 мес. Холодный старт, 3–5 ч/нед.

План: `docs/promo-plan-first-paying.md` в репо.

## Каналы/механики
- Автопостер ВК: очередь `vk_post_queue` (миграция 048) + cron `vkQueueJob`, постит из md-файла контента в сообщество `calendacha`. Нужен пользовательский VK-токен. Тексты — см. [[Документация/vk-content/batch-2026-07|ВК-контент: batch-2026-07]] (живая папка, не копия — подробнее [[12 Полный репозиторий (зеркало)]]).
- Тестеры: определяются по погашенному промокоду (список неполный, дрейфует). Перед маркетинговыми рассылками — `UPDATE is_test=true`.
- Демо-аккаунт: `demo@dacha.ru` / `demo1234` (прод, store=gplay, триал 30 дн), участок Краснодар с задачами на «Сегодня» каждого типа — для скриншотов.
