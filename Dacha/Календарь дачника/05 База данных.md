# База данных

PostgreSQL, БД `dacha_db`, пользователь `dacha_user`. Миграции: `backend/src/db/migrations/`, нумерация 001–048+.

> ⚠️ Полная цепочка `npm run migrate` на проде НЕ идемпотентна (падает на 009 — `must be owner of care_alert_log`). Новые миграции — точечно (Node dotenv+pg, либо `sudo -u postgres psql -d dacha_db -f` для чистых ALTER/UPDATE).

## Ключевые таблицы/поля
- `crops.climate_zones` — JSONB, USDA зоны 3–6
- `crops.watering_details` — JSONB `{frequency_days, notes}`
- `crops.fertilizing_schedule` — JSONB массив этапов
- `crops.diseases`, `crops.pests` — JSONB
- `crops.good_neighbors`/`bad_neighbors` — TEXT[]
- `gardens.climate_zone` — fallback через `getZoneForRegion(region)`
- `guide_entries` + `crop_guide_entries` — справочник проблем растений (~68 записей, 52/68 с фото)
- `payments`, `users.payment_method_id/auto_renew/plan` — ЮKassa
- `vk_post_queue` — автопостер ВК
